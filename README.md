# 安卓蓝牙机器人遥控 App

这是一个最小可用版本：
- 通过经典蓝牙 SPP 连接机器人（常见 HC-05/HC-06 模块）
- 发送运动指令：前进、后退、左转、右转、停止
- 通过 OTG 接入 UVC 视频采集卡显示 5.8G 模拟图传画面

## 指令协议
App 发送单字符 + 换行：
- `F\n` 前进
- `B\n` 后退
- `L\n` 左转
- `R\n` 右转
- `S\n` 停止

机器人端只要按行读取串口数据并解析这 5 种字符即可。

## 图传链路
当前图传方案适用于下面的硬件链路：
- 5.8G 模拟图传接收机
- AV 输出到 UVC 视频采集模块
- 手机通过 OTG 连接 UVC 采集卡

在 App 里点击左侧的“图传连接”即可打开 UVC 设备选择并开始预览。

## 使用步骤
1. 在系统蓝牙设置中，先把手机与机器人蓝牙模块配对。
2. 用 Android Studio 打开本项目。
3. 运行到真机，首次启动允许蓝牙权限。
4. 点击“选择设备”选择已配对的机器人蓝牙模块。
5. 点击“连接”，连接成功后点击方向按钮发送控制指令。
6. 点击左侧“图传连接”，选择 OTG 采集卡后即可在中间大屏查看实时画面。

## 兼容性说明
- `minSdk = 24`
- Android 12+ 使用 `BLUETOOTH_SCAN`/`BLUETOOTH_CONNECT` 运行时权限。
- 该示例默认使用 SPP UUID: `00001101-0000-1000-8000-00805F9B34FB`。
